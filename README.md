# ElfAnalyzer

## About
ElfAnalyzer is a toolkit for performing topological analysis of the
Electron Localization Function defined on a regular grid. It is designed primarily
for outputs from pseudopotential codes such as [VASP](https://www.vasp.at/).

It is part of my PhD at UNC Chapel Hill in the [Warren Lab](https://materials-lab.io/) with
funding from the [NSF's GRFP](https://nsfgrfp.org/).
